Selection Sort

Selection Sort is one of the simplest sorting algorithms, often taught as an introduction to sorting concepts. The idea is straightforward: repeatedly find the smallest element in the unsorted portion of the list and place it at the beginning. After the first pass, the smallest element is in its correct position. The process then continues with the remaining unsorted elements until the list is fully sorted. While easy to understand and implement, Selection Sort is not very efficient for large datasets, since it always performs O(nÂ²) comparisons regardless of the input order. However, it has the advantage of making at most n swaps, which can be useful when writing operations are costly. It is best suited for small datasets or when memory swaps must be minimized.
